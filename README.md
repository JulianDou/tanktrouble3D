Inspired by [https://tanktrouble.com](https://tanktrouble.com)

Game available at [https://juliandou.github.io/tanktrouble3D/](https://juliandou.github.io/tanktrouble3D/)

Player 1 controls :
ZQSD - Movement
A - Fire
F1 - Toggle AI

Player 2 controls :
Arrow Keys - Movement
Space - Fire
F2 - Toggle AI

Each player can shoot 5 times before having to reload (projectiles disappear after a while)
There are three different bonuses you can get in boxes
